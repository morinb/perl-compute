perl-compute
============

perl Math formula parser and calculator based on the Shunting-Yard Algorithm :
http://en.wikipedia.org/wiki/Shunting-yard_algorithm

Run 
```perl
perl Makefile.PL
```

Then on windows
```shell
 nmake install
 nmake test
```

_To install nmake see : http://johnbokma.com/perl/make-for-windows.html_
